# Piedra, Papel, Tijera, Lagarto, Spock

## ¿Como descargar e instalar la aplicación?

Para descargar la aplicación se tiene que clonar el repositorio, para ello, copiar el siguiente comando en consola

```
git clone https://github.com/francogarcino/unq-ui-franco-garcino-trabajo-final.git
```

Para instalar, entrar en la carpeta ```trabajofinal-ui```, que se encuentra dentro de la carpeta generada al clonar el repositorio, y ejecutar desde esa ubicación el comando

```
npm install
```

## Ahora, ¿como ejecutar la aplicación?

Para ejecutar la aplicación, se debe ejecutar el siguiente comando en la misma ubicación que el anterior

```
npm start
```

La aplicación se levantará de manera local en http://localhost:3000
